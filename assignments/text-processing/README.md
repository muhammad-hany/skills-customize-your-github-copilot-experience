# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice file I/O and text manipulation in Python by building a program that processes and analyzes text data. Students will learn to read/write files, use string methods, and implement parsing logic.

## 📝 Tasks

### 🛠️ Read and Clean Text Data

#### Description
Write a Python script that reads input text from a file, normalizes whitespace, removes punctuation, and converts text to lowercase.

#### Requirements
Completed program should:

- Read text from `input.txt`
- Remove punctuation characters (e.g., `. , ! ? ; :`) from the text
- Convert text to lowercase
- Strip extra whitespace and normalize spaces between words
- Write cleaned text to `cleaned.txt`

### 🛠️ Analyze Word Frequency

#### Description
Create a function to count word frequency from cleaned text and print the top 10 most common words.

#### Requirements
Completed program should:

- Read from `cleaned.txt`
- Count occurrences of each word
- Print the top 10 words and their counts in descending order
- Handle ties consistently (alphabetical order for equal counts)
- Include comments explaining the analysis steps
